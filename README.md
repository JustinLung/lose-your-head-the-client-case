## ⚡ Installatie & Gebruik
```
# Cloning the repository
git clone git@github.com:JustinLung/lose-your-head-the-client-case.git

# Moving to the folder
cd lose-your-head-the-client-case

# Install NPM packages
npm install

# Run local server
npm run start
```
